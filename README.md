# learning_git
## Creating new project:
```cmd
git init
git remote add origin <url of new repo>
git add .
git commit -am "message"
 git push --set-upstream origin master
 ```
 
 ## Pushng changes to repo-
```cmd
git add . # if you added a file
git commit -am "message"
 git push
 ``` 
 
 ## Pulling changes to local repo
 ```cmd
 git pull
 ```
